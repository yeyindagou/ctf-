# ctf-
小皮简单部署ctf竞赛平台(适合小白搭建）
简单部署
搭建前先下载个小皮面板
源码下载好先上传至www目录下
再打开数据库导入test.sql
最后把db.php里的数据库配置信息修改为你自己的数据库名、账号和密码

题目部署
